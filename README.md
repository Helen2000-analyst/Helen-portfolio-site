# Helen-Car-Pivot-Sales-Data-Analysis (Interactive Dashboard creation using MS Excel)

## Project Objective

Helen car sales company wants to create an annual sales report for 2024. So that, the owner will understand their customers and grow more sales in the coming year.

## Dataset used

- <a href= "https://d.docs.live.net/1e3a355d72250c12/Documenti/CAR%20PIVOT.xlsx">Dataset</a>

## Questuons [KPI's] 

-1.	Total Price by Make
 Create a pivot table showing the total price of cars grouped by the Make.

-2.	Average Cost by Model
 Calculate the average cost of cars grouped by Model.

-3.	Sum of Mileage by Color
 Summarize the total mileage for cars grouped by their Color.

-4.	Average Price by Make and Color
  Display the average price for each combination of Make and Color.

 -5.	Count of Cars by Make and Model
  Count how many cars of each Make and Model are available.

-6.	Total Cost by Make and Model
 Sum the total cost of cars grouped by Make and Model.

-7.	Maximum Mileage by Color
 Find the car with the maximum mileage for each Color.

-8.	Total Cost and Price by Make
 Show a pivot table with both Price and Cost summed for each Make.

-9.	Average Mileage by Model and Color
 Calculate the average mileage for each Model and Color.

-10.	Count of Cars by Color
 Count how many cars of each Color are in the dataset.

-11.	Min and Max Price by Make and Model
 Show the minimum and maximum price of cars grouped by Make and Model.

-12.	Sum of Price by Make with Filters on Color
 Create a pivot table that sums the price by Make, and filter it by specific Color values.

-13.	Total Mileage by Make and Color
 Summarize the total mileage grouped by Make and Color.

-14.	Percentage Contribution of Each Make to Total Price
 Show what percentage of the total price each Make contributes.

-15.	Average Cost by Make and Filter by Price > $3,000
Filter the data to only show cars with a price greater than $3,000 and display the average cost grouped by Make.



## Process

- Verified data for any missing anomalities, and sort them out.

- Made sure data is consistent and clean with respect to data type, data format, data values used.

- Created pivot tables according to the question asked.

- Merged all pivot tables into onedashboard and applied slicer to make it dynamic.
